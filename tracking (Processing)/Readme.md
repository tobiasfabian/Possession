A Processing script which recognizes human from the picture of the Kinect and is sending data to `ar-drone-kinect` via WebSockets.

# Fly baby, fly!
Make sure you run `jsonJoints.pde` (to start the WebSockets server) before you run the `ar-drone-kinect/index.js`.

## Libraries
* [simple-openni](https://code.google.com/p/simple-openni/wiki/Installation)
* [WebSocketP5](https://github.com/muthesius/WebSocketP5)
